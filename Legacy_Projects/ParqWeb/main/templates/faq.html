Frequently Asked Questions

Where can I download the Parq app?

You can download the app from the Android Market.

What is included with the app downloaded from the Android Market?

The App is a "one-touch" parking system. It allows a user to start their parking with the touch of a button. Furthermore, the App serves as a reminder of how much time is left, sends reminders when the meter is running out, and allows for a user to refill the meter wherever they are. Finally, the App also acts as a tool to locate the user's car when they are ready to leave.

How do I set reminders in the app?

The settings menu allows a user to set how often they would like to receive reminders.

How do I add more time?

You can select more time using the Refill function and then pay for the extra time.

Can I leave even if there is time remaining?

Yes, but you will not receive any discount for doing so.

What happens when the time is up?

If time runs low, you will receive a reminder. If you choose not to refill, then the time will run out and you will be susceptible to receiving a ticket.

Can I set automatic payments?

Yes, there is an option under the Refill option for automatic refills. You can set how much time to have left before an automatic refill as well as how many automatic refills you want.
